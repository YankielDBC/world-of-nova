# World of Nova - Encyclopedia (v0)

## Purpose
This file is the compact reference index for names used in the game UI.
It is intentionally short so we can keep in-game lines mobile-friendly and link details here later.

## Skills
- `gather`: basic collection and forage actions.
- `chop`: woodcutting and harvest actions.
- `mine`: mining and extraction actions.

## Tool Catalog
- `canapez`: Cana de Bambu (`fishing`)
- `hachaPiedra`: Hacha de Piedra (`woodcutting`)
- `picoPiedra`: Pico de Piedra (`mining`)
- `basket`: Canasta de Paja (`gathering`)
- `varaMadera`: Vara de Madera (`harvesting`)

## Biomes
- `forest`
- `swamp`
- `plains`
- `river`
- `volcano`

## Place Archetypes
- `FIXED`: permanent map place linked to coordinates.

## Rarity Codes (UI)
- `C`: common
- `U`: uncommon
- `R`: rare
- `E`: epic
- `L`: legendary

## Ground Loot
- `resource`: dropped stackable resources.
- `tool`: dropped tool instances.
- `bag`: dropped bag items.
